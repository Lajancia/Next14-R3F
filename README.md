# Next14 + React Three Fiber

## How to Run?
```
npm install
npm run panda
npm run dev
```

## Added features
### Main Home Page
* Macro Keyboard

### Gallery Page
* Images display

### About Page
* Skills
* Career
* blog post
* work history

### Other
* Next-translation
* page transition
* initial page loading
